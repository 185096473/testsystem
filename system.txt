项目构建规范：
1.对应文件放入对应文件夹
  1.1样式中必须要有一个重置样式表
2.每一个项目必须存在一个文件index.html
  注：所有服务器访问文件时必须首先要找到index.html当做默认打开文件
3.项目管理  git 或 svn
4.优化项目
5.传到服务器 ftp工具  filezilla(推荐使用)  8uftp

项目开发流程：  前端岗位来说：
1.理解需求  项目经理  产品经理 
2.拿到设计稿  ui    
   2.1 和ui对接需求  页面效果及功能
3.制作网页 （静态）
4.进行数据交互   后端  php  java
5上传服务器
 5.1首先上传到测试服务器  
    5.1.1 测试人员进行测试
    5.1.2 调bug 重复测试
    5.1.3 测试通过 
 5.2上传到正式服务器   （外网）
    5.2.1调bug
